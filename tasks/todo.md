# Tasks & Maintenance

## Brand Icons

When you update `public/icon.png`, run:

```bash
npm run icons
```

This regenerates all icon assets in `public/icons/` and copies `favicon.ico` to `src/app/`:

- favicon.ico, favicon-16x16.png, favicon-32x32.png
- apple-touch-icon.png (180x180)
- icon-192x192.png, icon-512x512.png (PWA)
- og-image.png (1200x630, social sharing)
- safari-pinned-tab.svg
