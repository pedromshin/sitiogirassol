**2. Listing Settings Changes**

Section A -- Minimum Stay (LISTING-03, per D-13):
- Steps: Airbnb Host Dashboard -> Listing -> Availability -> Trip length -> Minimum night stay
- Set to: 2 nights
- Note: For weekday gap-fills (e.g., a single open Monday between two bookings), the owner can temporarily reduce to 1 night for that specific date range via Calendar -> select dates -> Edit -> Minimum stay override
- Why: 1-night stays have the lowest revenue per cleaning cycle. With R$180 cleaning fee, 1-night at R$280 yields only R$100 after cleaning (R$84 net after Airbnb 16% fee).

Section B -- Advance Notice (LISTING-04, per D-14):
- Steps: Airbnb Host Dashboard -> Listing -> Availability -> Advance notice
- Set to: 1 day (minimum recommended; 2 days if owner needs more prep time)
- Why: Same-day bookings don't allow time to prepare the property. 1-day notice gives time to arrange cleaning and setup.

Section C -- Categories (LISTING-05, per D-15):
- Steps: Airbnb Host Dashboard -> Listing -> Property and rooms -> Property type and categories
- Verify these categories are selected: Countryside, Tropical
- Also check: Property type should be "Farm stay" or "Entire home" (whichever best describes the sitio)
- Additional relevant categories to check for: "Pools", "Amazing views" (if available in the Airbnb category list for this region)
