## 2. Base Pricing Changes

Set the standard nightly rates for weekdays and weekends.

**Weekday rate (Sunday through Thursday): R$280/night**
**Weekend rate (Friday and Saturday): R$380/night**

### How to set base pricing on Airbnb:

**Option A -- If Airbnb offers separate weekday/weekend pricing:**
1. Go to Airbnb Host Dashboard -> Listing -> Pricing
2. Set base price to R$280/night
3. Look for "Weekend pricing" or "Custom pricing rules"
4. Set Friday and Saturday to R$380/night

**Option B -- If only one base price field is available:**
1. Set base price to R$280/night
2. Go to Calendar view
3. For each Friday and Saturday over the next 3 months, click the date and set a custom price of R$380/night
4. This takes about 15-20 minutes but ensures correct weekend pricing

**The math:** R$380 weekend rate is 35.7% above the R$280 weekday base, satisfying the 25-35% weekend premium target (D-05).

## 3. Discount Reductions

Reduce the length-of-stay discounts that have been crushing ADR from R$340 to R$178.

### Changes:
- **Weekly discount:** Change from **33%** to **12%**
- **Monthly discount:** Change from **50%** to **27%**

### How to change discounts:
1. Go to Airbnb Host Dashboard -> Listing -> Pricing
2. Scroll to "Length-of-stay discounts" (or "Discounts")
3. Set weekly discount to **12%**
4. Set monthly discount to **27%**
5. Save

### Verification (do this after changing):

After saving the new discount percentages, preview your listing as a guest to verify correct pricing:

**7-night weekday stay verification:**
- Calculation: R$280 x 7 nights x (1 - 0.12) = R$280 x 7 x 0.88 = R$1,725 subtotal
- Plus cleaning fee: R$1,725 + R$180 = **R$1,905 total** (before Airbnb guest service fee)
- Effective nightly rate: R$246/night

**28-night stay verification:**
- Calculation: R$280 x 28 nights x (1 - 0.27) = R$280 x 28 x 0.73 = R$5,722 subtotal
- Plus cleaning fee: R$5,722 + R$180 = **R$5,902 total** (before Airbnb guest service fee)
- Effective nightly rate: R$204/night

### Warning (Pitfall 6):
After changing discounts, check for **stacked promotions**. Airbnb may have active "new listing" discounts or promotional offers that stack on top of your length-of-stay discounts, making the final price even lower than expected. To check:
1. Go to Listing -> Pricing -> Promotions
2. Disable any active promotions that stack with length-of-stay discounts
3. Preview as guest for 7-night and 28-night stays to confirm prices match the calculations above

## 4. Cleaning Fee

Add a cleaning fee to discourage low-value 1-night stays and recover actual cleaning costs.

**Cleaning fee: R$180**

### How to add cleaning fee:
1. Go to Airbnb Host Dashboard -> Listing -> Pricing
2. Scroll to "Fees" section
3. Find "Cleaning fee" field
4. Enter **180** (R$180)
5. Save

**Purpose:** The cleaning fee is a flat charge regardless of stay length. This means:
- 1-night stay: R$180 cleaning fee on R$280 = R$460 total (cleaning is 39% of bill -- discourages short stays)
- 2-night stay: R$180 cleaning fee on R$560 = R$740 total (cleaning is 24% of bill)
- 7-night stay: R$180 cleaning fee on R$1,960 = R$2,140 total (cleaning is 8% of bill -- minimal impact on longer stays)

This naturally incentivizes longer stays without requiring large length-of-stay discounts.
