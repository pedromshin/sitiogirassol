## 1. Listing Title Options (Portuguese)

Airbnb limits titles to **50 characters**. On mobile devices, only the **first 32 characters** are visible. The most important keywords must appear within those first 32 characters.

Required keywords (per D-12): sitio, natureza, piscina, churrasco, Sorocaba, Sao Paulo. Not all 6 fit in 50 characters -- front-load the highest-conversion keywords (sitio + piscina) and put the rest in the description.

### Option A (RECOMMENDED)

```
Sitio c/ Piscina e Natureza | Sao Roque SP
```

- **Total:** 44 characters
- **Mobile shows:** `Sitio c/ Piscina e Natureza | S` (32 characters)
- **Keywords included:** sitio, piscina, natureza, Sao Roque, SP (Sao Paulo state)
- **Why recommended:** Front-loads the two highest-conversion keywords (sitio + piscina) within mobile cutoff. "Natureza" adds emotional appeal. Location reference "Sao Roque SP" provides geographic context. Clean formatting with pipe separator.

### Option B

```
Sitio Girassol: Piscina e Natureza em SP
```

- **Total:** 41 characters
- **Mobile shows:** `Sitio Girassol: Piscina e Nature` (32 characters)
- **Keywords included:** sitio, piscina, natureza, SP (Sao Paulo state)
- **Note:** Uses the property brand name "Girassol" which builds recognition but takes up characters. Fewer location keywords than Option A.

### Option C

```
Sitio c/ Piscina, Churrasqueira | Sorocaba
```

- **Total:** 44 characters
- **Mobile shows:** `Sitio c/ Piscina, Churrasqueira` (32 characters -- exact fit)
- **Keywords included:** sitio, piscina, churrasco (as churrasqueira), Sorocaba
- **Note:** Includes the churrasco keyword directly in the title and references Sorocaba. Trades "natureza" for "churrasqueira" -- good if BBQ is the stronger selling point for the target audience.

### Title Comparison

| | Option A | Option B | Option C |
|---|---------|----------|----------|
| Characters | 44 | 41 | 44 |
| Mobile-safe keywords | sitio, piscina, natureza | sitio, piscina | sitio, piscina, churrasqueira |
| Location in title | Sao Roque SP | SP | Sorocaba |
| Brand name | No | Yes (Girassol) | No |
| Emotional appeal | Nature | Brand | BBQ |

**Recommendation:** Use **Option A**. It has the best balance of conversion keywords, emotional appeal ("natureza"), and geographic specificity ("Sao Roque SP"). The nature/pool combination is the strongest hook for Sao Paulo guests looking for weekend escapes.

## How to Update on Airbnb

1. Go to **Airbnb Host Dashboard**
2. Click **Listing**
3. Click **Listing details**
4. Find **Title** field -- paste the recommended title (Option A)
5. Find **Description** field -- paste the Portuguese description
6. If Airbnb shows a separate English description field, paste the English version there
7. Click **Save**
8. Preview the listing to verify the title displays correctly on desktop and mobile
