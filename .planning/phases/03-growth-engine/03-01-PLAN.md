**Document structure (sections in this order):**

1. **Strategy Overview** - Explain the 3-layer funnel concept: Awareness feeds Consideration feeds Conversion. Include a visual funnel diagram using ASCII art showing Layer 1 (cold audience, 60% budget) -> Layer 2 (warm audience, 25% budget) -> Layer 3 (hot audience, 15% budget). Per D-04.

2. **Prerequisites** - What must exist from Phase 2 before starting:
   - Custom Audience "Website Visitors - 180d" (created in Phase 2 Step 2)
   - Custom Audience "Instagram Engagers - 180d" (created in Phase 2 Step 2)
   - Messages_WhatsApp_v1 campaign (created in Phase 2 Step 3 -- this becomes Layer 3)
   - Facebook Pixel active on sitiogirassol.org
   - WhatsApp Business account linked to Meta

3. **Step 1: Check Source Audience Sizes** - Per D-11, instructions to navigate to Ads Manager > Audiences and check the size of "Website Visitors - 180d" and "Instagram Engagers - 180d". If either is >= 1,000, proceed to Step 2 (lookalike creation). If both are < 1,000, skip to Step 3 and use interest-based targeting as interim. Include exact navigation path in Meta Ads Manager.

4. **Step 2: Create Lookalike Audiences** - Per D-08, D-09, D-10:
   - Create "Lookalike - Website Visitors 1-3% Brazil" from "Website Visitors - 180d" source, 1-3% similarity, location: Brazil. Full step-by-step with exact field values.
   - Create "Lookalike - Instagram Engagers 1-3% Brazil" from "Instagram Engagers - 180d" source, 1-3% similarity, location: Brazil. Full step-by-step.
   - Include warning about Meta's lookalike auto-expansion behavior (from Research Pitfall 2): since late 2024, Meta auto-expands lookalikes beyond specified percentage for performance goals. The 1-3% is a starting signal, not a hard boundary. Monitor CPM and reach.
   - Verification checklist: both audiences show "Ready" status in Audiences tab.

5. **Step 3: Create Layer 1 Campaign (Awareness)** - Per D-01, D-06:
   - Campaign name: "Awareness_Reels_v1"
   - Objective: Awareness (or Reach)
   - Budget: CBO enabled, daily budget = 60% of total daily spend. At R$20/day total = R$12/day. At R$25/day total = R$15/day.
   - Create 3 ad sets:
     - Ad Set 1: "LAL_WebVisitors" -- Lookalike - Website Visitors audience (skip if source < 1,000; use interest-based instead)
     - Ad Set 2: "LAL_InstaEngagers" -- Lookalike - Instagram Engagers audience (skip if source < 1,000)
     - Ad Set 3: "Interest_SP" -- Interest-based targeting as baseline (always active). Location: Sao Paulo state. Interests: "viagem", "turismo", "natureza", "aluguel de temporada", "piscina", "churrasco", "fim de semana". Age: 25-55. Gender: All.
   - Creative: Repurpose top-performing Instagram Reels as ad creative. Instructions to find best-performing Reels in Instagram Insights (sort by Reach), download them, and upload as ad creative. Per D-01.
   - Ad copy (Portuguese): Primary text example: "Fuja da rotina. Sitio com piscina, churrasqueira e muita natureza a 1h de SP. Reservas pelo link ou WhatsApp." Headline: "Sitio Girassol - Seu Refúgio Pertinho de SP". CTA: "Send Message" or "Learn More".
   - CRITICAL WARNING: Do NOT select "Housing" as Special Ad Category. Frame ads as tourism/experience. Use "reservar" (book) and "visitar" (visit), NEVER "alugar" (rent) or "morar" (live). Include the appeal process if ads get flagged (from Research Pitfall 1).
   - Placements: Advantage+ placements (let Meta optimize).
   - Verification checklist: Campaign shows "Active" status, 3 ad sets visible, CBO toggle is ON.

6. **Step 4: Create Layer 2 Campaign (Consideration)** - Per D-02, D-06:
   - Campaign name: "Consideration_Retarget_v1"
   - Objective: Engagement (or Traffic)
   - Budget: CBO enabled, 25% of total daily spend. At R$20/day = R$5/day.
   - Launch timing: Start 1 week AFTER Layer 1 begins (Layer 1 needs time to generate engaged users to retarget).
   - Create 3 ad sets:
     - Ad Set 1: "VideoViewers_50pct" -- Custom Audience of people who watched >50% of any video from Layer 1 campaigns (instructions to create this Custom Audience: Audiences > Custom Audience > Video > People who watched at least 50% > select Layer 1 ad videos > 30 days)
     - Ad Set 2: "InstaEngagers_Recent" -- Use existing "Instagram Engagers - 180d" Custom Audience
     - Ad Set 3: "WebVisitors_Recent" -- Use existing "Website Visitors - 180d" Custom Audience
   - Creative: Property detail carousels (multiple images showing rooms, pool, BBQ, nature), testimonial quote graphics, comparison posts ("Hotel vs Sitio Girassol"). Different creative from Layer 1 -- more detail-oriented, less awareness-oriented.
   - Ad copy: More detail-oriented. Example: "Piscina, churrasqueira, 4 quartos e natureza exuberante. Ate 15 hospedes. A partir de R$280/noite. Fale conosco pelo WhatsApp." Headline: "Sitio Completo para Seu Grupo". CTA: "Send Message".
   - Verification checklist: Campaign shows "Active" (or "Scheduled"), 3 ad sets configured, CBO ON.

7. **Step 5: Restructure Phase 2 Campaign as Layer 3 (Conversion)** - Per D-03:
   - Do NOT create a new campaign. Restructure the existing "Messages_WhatsApp_v1" campaign from Phase 2.
   - Rename campaign to "Conversion_Messages_v1".
   - Adjust budget: CBO at 15% of total daily spend. At R$20/day = R$3/day. WARNING: R$3/day is below Meta's recommended R$10/day minimum for Messages campaigns. If the campaign shows "Learning Limited" status for 5+ consecutive days or receives fewer than 1 message/day for 5 days, reallocate to 50/25/25 split (take from Layer 1).
   - Pause the cold-audience ad set from Phase 2 (keep only retargeting ad sets that target warm audiences).
   - Optionally add a new ad set targeting Layer 2 engagers: Custom Audience from people who engaged with Layer 2 ads.
   - Launch timing: Start 2 weeks AFTER Layer 1 begins.
   - Creative: Direct CTA to WhatsApp. Price-anchored messaging. Example: "Ja conhece o Sitio Girassol? Reserve seu fim de semana a partir de R$280. Fale comigo agora pelo WhatsApp!" Headline: "Reserve Agora - Vagas Limitadas". CTA: "Send WhatsApp Message".
   - Include the Layer 3 budget starvation mitigation from Research Pitfall 3.
   - Verification checklist: Campaign renamed, budget adjusted, cold ad set paused, status "Active" or "Scheduled".

8. **Step 6: Staggered Launch Timeline** - Visual timeline:
   - Week 1: Launch Layer 1 only (R$12/day)
   - Week 2: Launch Layer 2, keep Layer 1 running (R$12 + R$5 = R$17/day)
   - Week 3: Restructure Layer 3, all 3 layers active (R$12 + R$5 + R$3 = R$20/day)
   - Explain WHY staggered: Layer 1 needs to accumulate video viewers and engagers before Layer 2 has anyone to retarget; Layer 2 needs to warm people up before Layer 3 converts them.

9. **Step 7: Weekly Performance Review** - Per D-07:
   - Create a review checklist table with metrics per layer:
     - Layer 1: CPM (target < R$30), Reach (growing week over week), Video views >50% (feeding Layer 2)
     - Layer 2: CPC (target < R$1.00), Engagement rate (target > 2%), Website clicks
     - Layer 3: CPL/Cost per message (target < R$5.00), Messages received, Bookings attributed
   - Decision framework: When to adjust budgets (shift from 60/25/15 toward underperforming layer), when to refresh creative (after 2-3 weeks or when frequency > 3), when to pause an ad set (CTR < 0.5% after 1 week).
   - Include the anti-pattern warning: do NOT set and forget. Review every week, adjust every 2 weeks.

10. **Troubleshooting Section** - Cover the 3 critical pitfalls from Research:
    - Housing Special Ad Category flag: how to avoid, how to appeal
    - Lookalike auto-expansion: monitoring and mitigation
    - Layer 3 budget starvation: reallocation strategy
    - Include a "What to do if your ads are rejected" section.

11. **Seasonal Considerations** - Per Research seasonal context:
    - April (shoulder): Push Easter availability (Apr 17-21) in Layer 3 creative
    - May-Sep (low season): Shift ad copy to "winter getaway" / "fuja do frio" messaging
    - Update creative seasonally -- golden-hour pool shots for spring/summer, cozy/fire content for winter
